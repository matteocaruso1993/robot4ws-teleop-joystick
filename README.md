# robot4ws-teleop-joystick

## Introduction
This package allows to telemanipulate the rover Archimede by using the Logiteck **xyz** bluetooth controller.

## Installation
```
sudo apt-get install ros-$ROS_DISTRO-joy
git clone ...
catkin_make
```
